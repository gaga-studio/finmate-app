# finmate 디자인 가이드

finmate는 20대 사용자가 자신의 금융 생활을 부담 없이 확인하고, 작은 행동으로 이어가도록 돕는 모바일 웹앱이다. 전체 인상은 **깔끔함, 말랑한 친근함, 금융 서비스다운 신뢰감**을 함께 가져간다.

## 디자인 키워드

- **Fresh Money Mate**: 딱딱한 금융 앱보다 산뜻하고 가벼운 동반자 느낌
- **Claymorphism UI**: 부드럽게 눌린 듯한 입체감, 말랑한 카드, 은은한 그림자
- **Clean Youthful**: 20대가 촌스럽다고 느끼지 않도록 장식은 줄이고 여백과 타이포로 정리
- **Action First**: 소비, 저축, 투자 데이터를 "봐야 할 것"보다 "바로 할 수 있는 것"으로 보여주기

## 메인 컬러

메인 컬러는 `#cdf1eb`를 사용한다. 밝은 민트 계열이라 앱 전체의 배경, 카드 하이라이트, 긍정 상태에 잘 어울린다. 단, 너무 연해서 텍스트 색상으로 직접 쓰기보다는 배경과 표면 컬러로 사용한다.

| 역할 | 색상 | 사용처 |
|---|---|---|
| Primary Mint | `#cdf1eb` | 핵심 배경, 활성 카드, 저축/긍정 흐름 |
| Primary Deep | `#1f8f83` | CTA, 선택 상태, 아이콘 강조 |
| Primary Dark | `#11615c` | 중요한 숫자, 그래프 포인트 |
| Mint Light | `#effbf8` | 전체 앱 배경, 빈 상태 배경 |
| Mint Soft | `#dff7f2` | 보조 카드, 섹션 배경 |
| Ink | `#17211f` | 본문 텍스트 |
| Muted Ink | `#667571` | 보조 텍스트, 설명 |
| Surface | `#f8fffd` | 카드 기본 면 |
| White | `#ffffff` | 최상단 표면, 모달 |
| Line | `#d8ebe7` | 구분선, 입력 필드 보더 |
| Warning | `#ffb84d` | 주의, 예산 초과 직전 |
| Danger | `#ff6b6b` | 초과, 실패, 삭제 |
| Violet | `#8b7cf6` | 투자 지표 보조색 |
| Blue | `#69a7ff` | 소비 지표 보조색 |

## 컬러 사용 원칙

- 앱의 큰 배경은 `#effbf8` 또는 `#f8fffd`를 사용해 밝게 유지한다.
- `#cdf1eb`는 카드 표면, 활성 탭, 그래프 면적, 미션 완료 배경처럼 넓은 면에 사용한다.
- 버튼과 선택 상태는 `#1f8f83`처럼 더 진한 민트/틸을 사용해 클릭 가능성을 명확히 한다.
- 소비, 저축, 투자는 서로 구분되도록 보조색을 둔다.
  - 소비: `#69a7ff`
  - 저축: `#1f8f83`
  - 투자: `#8b7cf6`
- 빨간색은 경고와 손실에만 제한적으로 사용한다.

## 타이포그래피

추천 폰트는 **Pretendard**다. 이미 프로젝트에도 포함되어 있고, 한국어 모바일 UI에서 가장 안정적으로 읽힌다. 20대 타겟에게도 익숙하고 깔끔하며, 금융 정보의 숫자 가독성도 좋다.

대안이 필요하면 아래 순서를 권장한다.

| 우선순위 | 폰트 | 이유 |
|---|---|---|
| 1 | Pretendard | 한국어 UI에 최적화, 깔끔함, 숫자 가독성 우수 |
| 2 | SUIT | 조금 더 부드럽고 젊은 인상 |
| 3 | Noto Sans KR | 안정적이지만 덜 개성적 |
| 4 | Paperlogy | 헤드라인용으로만 사용 시 산뜻함 |

권장 조합:

- 기본 UI: `Pretendard`
- 숫자/금액: `Pretendard`, `font-variant-numeric: tabular-nums`
- 큰 캠페인 문구나 Wrapped 카드 제목: `Pretendard` 700~800 또는 `Paperlogy` 선택 사용

## 글자 크기

| 역할 | 크기 | 굵기 | 사용처 |
|---|---:|---:|---|
| Display | 28~32px | 800 | Wrapped 카드, 주요 성취 문구 |
| Page Title | 22~24px | 750 | 탭별 상단 제목 |
| Section Title | 18~20px | 700 | 카드 묶음 제목 |
| Card Title | 16~17px | 700 | 카드 내부 주요 라벨 |
| Body | 14~15px | 500 | 기본 본문 |
| Caption | 12~13px | 500 | 보조 설명, 날짜, 태그 |
| Amount | 24~36px | 800 | 금액, 퍼센트, 핵심 지표 |

## Claymorphism UI 방향

클레이모피즘은 과하게 3D처럼 보이면 장난감 같아질 수 있다. finmate에서는 **부드럽지만 정돈된 클레이 카드**로 적용한다.

### 기본 표면

```css
background: linear-gradient(145deg, #ffffff 0%, #e9faf6 100%);
border: 1px solid rgba(255, 255, 255, 0.7);
box-shadow:
  10px 14px 28px rgba(31, 143, 131, 0.16),
  -8px -8px 20px rgba(255, 255, 255, 0.9),
  inset 1px 1px 0 rgba(255, 255, 255, 0.85);
border-radius: 24px;
```

### 눌림 상태

```css
box-shadow:
  inset 5px 5px 12px rgba(31, 143, 131, 0.16),
  inset -5px -5px 12px rgba(255, 255, 255, 0.85);
transform: translateY(1px);
```

### 떠 있는 CTA

```css
background: linear-gradient(145deg, #2fb3a5 0%, #1f8f83 100%);
color: #ffffff;
box-shadow:
  0 12px 24px rgba(31, 143, 131, 0.28),
  inset 1px 1px 0 rgba(255, 255, 255, 0.32);
border-radius: 18px;
```

## 컴포넌트 규칙

### 카드

- 카드 반경은 20~28px 사이를 기본으로 한다.
- 정보 카드가 너무 귀여워 보이지 않도록 내부 정렬은 단정하게 유지한다.
- 카드 안에 카드가 겹치는 구조는 피하고, 필요한 경우 얇은 구분선이나 배경 톤 차이로 그룹을 나눈다.
- 핵심 숫자는 크게, 설명은 짧게 둔다.

### 버튼

- 주요 액션은 진한 틸 배경에 흰색 텍스트를 사용한다.
- 보조 액션은 밝은 민트 표면과 진한 텍스트를 사용한다.
- 아이콘 버튼은 원형 또는 정사각형에 가까운 형태로 만든다.
- 탭, 칩, 필터는 눌림 상태가 명확해야 한다.

### 그래프

- 선과 면은 부드러운 곡선을 사용한다.
- 그래프 배경은 과한 격자보다 옅은 라인과 하이라이트 포인트를 사용한다.
- 저축/성장 그래프는 민트, 소비는 블루, 투자는 바이올렛으로 구분한다.

### 입력/선택

- 입력 필드는 흰색 또는 아주 연한 민트 배경을 사용한다.
- 포커스 상태는 `#1f8f83` 보더와 은은한 외곽 그림자로 표현한다.
- 슬라이더, 토글, 세그먼트 컨트롤은 말랑한 표면감과 부드러운 이동 애니메이션을 준다.

## 레이아웃

- 모바일 우선: 390×844 뷰포트에서 가장 자연스럽게 보이도록 설계한다.
- 주요 콘텐츠는 한 화면에 한 가지 메시지만 강하게 보이게 한다.
- 상단은 가볍게, 중앙은 핵심 지표, 하단은 다음 행동으로 이어지는 구조를 권장한다.
- 여백은 16px, 20px, 24px 단위 중심으로 사용한다.
- 탭바는 항상 명확하고 손가락으로 누르기 쉬운 크기를 유지한다.

## 모션

- 모션은 "말랑한 반응"을 주되, 데이터 확인을 방해하지 않아야 한다.
- 카드 전환: 220~360ms
- 버튼/칩 피드백: 120~180ms
- 차트 변화: 400~700ms
- 스프링은 살짝 탄성 있게 사용하되 과도한 튐은 피한다.

## 이미지/일러스트 방향

- Wrapped 카드와 공유 이미지는 앱의 즐거움을 담당하므로 조금 더 선명한 색을 사용해도 된다.
- 기본 UI 안의 이미지는 민트, 블루, 바이올렛 팔레트와 어울리는 톤을 유지한다.
- 금융 앱이지만 차갑고 딱딱한 은행 이미지보다 라이프스타일, 목표, 습관의 이미지를 우선한다.

## 피해야 할 것

- 민트색만 반복되어 화면 전체가 단조로워지는 것
- 과도한 그림자로 카드가 무겁거나 촌스럽게 보이는 것
- 너무 둥근 말풍선형 UI가 많아져 유아적인 인상이 나는 것
- 작은 보조 텍스트를 연한 민트 위에 낮은 대비로 올리는 것
- 금융 손실/경고 색을 장식 목적으로 사용하는 것

## 한 줄 기준

finmate의 디자인은 **밝은 민트 기반의 클레이모피즘 표면 위에, 20대가 매일 열어도 부담 없는 깔끔한 금융 행동 UI**를 만드는 것을 목표로 한다.
