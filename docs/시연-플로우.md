# FinMate 시연 플로우 (촬영 대본)

> 목표 서사: **"피드에서 ETF 하는 메이트를 발견 → 나도 해보고 싶다 → AI가 안전한 플랜을 짜주고 → 미션으로 실행"**
> 모든 수치는 고정(오늘 = 7/23 목 21:00): 오늘 예산 31% · 저축 45% · 투자 +12.7% · 총자산 1,400만 → 12월 예상 1,637만.
> 상태 초기화는 새로고침이면 충분 (모든 상태가 메모리 시드 기반).

---

## 1막 — 피드: 발견 (탭: 피드)

| # | 조작 | 화면에서 보이는 것 |
|---|---|---|
| 1 | 피드 탭에서 시작, 스토리 그리드 천천히 스크롤 | 12명의 각기 다른 그림체 스토리, 썸네일 우상단 ♥ 좋아요 뱃지, 그룹 카드(소득 유사·소비 유사·서울 20대·팔로잉) |
| 2 | (선택) 그룹 카드 1~2개 탭 | 민트 링+틴트로 선택 표시, 스토리가 조건 AND로 필터링. 다시 탭해 해제 |
| 3 | **곰손재테크의 "바이오 ETF, 이번 주 첫 매수!" 스토리** 탭 | 카드 확대 — "커피 3번 참아서 5만원 투자했어요", KODEX 바이오 시세 행 |
| 4 | 하단 닉네임 필 탭 → **[프로필]** 버튼 | 인라인으로 프로필/팔로우 버튼 등장 → 프로필 이동 |

## 2막 — 메이트 프로필: 진단의 시작 (곰손재테크)

| # | 조작 | 화면 |
|---|---|---|
| 5 | 프로필 진입 | 클레이 아바타 + "#자동이체 마스터" 뱃지, **소비 카드가 먼저** (오늘의 예산 34%) |
| 6 | 지표 탭(소비/저축/투자)·기간 칩 몇 개 전환 | 카드가 %·구간만 노출 (🔒 카테고리·구간만 공개) — 프라이버시 필터 어필 포인트 |
| 7 | **[나와 비교하기]** 탭 | 화면이 위로 슬라이드되며 비교 모드 — 카드가 "지혜 VS 곰손재테크" 대결 구도(좌=나) |
| 8 | 칩으로 뷰 2~3개 전환 | 좌우 리스트가 9뷰 동시 전환 (소비 탑5↔탑3, 소득 출처, 자산 구성…) |
| 9 | **[분석 비교]** 탭 | 로딩 도트 → AI 분석: 3판 대결표, 스타일 비교, **핵심 차이 "개별주 연습 → ETF 첫 매수"**, AI 처방 "30만원을 먼저 모으면…" ← 다음 막의 복선 |

## 3막 — 분석 탭: AI 코치와 플랜 (핵심 시퀀스)

| # | 조작 | 화면 |
|---|---|---|
| 10 | 분석 탭 이동 | 총자산 시뮬레이션 차트(현재 1,400만 → 12월 예상 1,637만, 9월 살짝 꺾였다 회복하는 곡선) + AI 코치 총평 |
| 11 | 총평 카드 **[자세히]** 탭 | 번호 타임라인 3줄 펼침 (지출·파리 자금·포트) |
| 12 | **[메이트/그룹 비교]** → **[그룹 고르기]** → 시트에서 **서울 20대 평균** 선택 | 시트 자동 닫힘 → 차트에 빨간선 겹침 — **내 초록선이 평균 위** ("서울 20대 평균보다 내 기울기가 가팔라요") |
| 13 | "시뮬레이션이 생성되었어요! ✨" → **[이어서하기]** | "추가 소비/저축 계획이 있으신가요? 예: 30만원 모아서 ETF 시도 해보고 싶어" |
| 14 | 입력창에 **`ETF 시도 해보고 싶어`** 타이핑 → 전송 | ⚠️ "지금 자금 없이 바로 시작하는 건 위험해 보여요" |
| 15 | (자동) | **차트가 30만원 플랜으로 전환** — "3개월 뒤 30만원 달성 · 월 10만원 × 3개월" + "먼저 30만원부터 모아볼까요?" |
| 16 | (자동) | "이 플랜, 실행할 가장 쉬운 방법 3가지" — 🎯절약 미션 / 💳카드 추천 / 🎁혜택 추천 |
| 17 | **[카드 추천]** 탭 | 하나카드 영하나 플러스(체크) 상세 카드 — 지출 관리 프레임 |
| 18 | **[혜택 추천]** 탭 | **하나증권 첫 거래 이벤트** — ETF 수수료 우대·투자지원금 (하나금융 연계 어필) |
| 19 | **[절약 미션]** 탭 | "30만원 모으기, 바로 실행 미션으로 만들어봤어요" + 미션 카드 |
| 20 | 미션 카드의 **[미션 탭에서 담기]** 탭 | → 미션 화면으로 이동 |

## 4막 — 미션: 실행 (탭: 미션)

| # | 조작 | 화면 |
|---|---|---|
| 21 | 진행 중인 미션 확인 | **딱 2개**: 이번 주 저축 5만원 추가(+120P) / 일 소비 예산 5% 줄이기(+80P) |
| 22 | 아래로 스크롤 → 추천 미션 | 상단 2개가 ETF 세트: **ETF 첫 시도 자금 30만원 모으기(+150P)** / **ETF 시작 전 투자 퀴즈(+60P)** |
| 23 | 두 개 모두 **[+담기]** | 토스트와 함께 진행 중으로 이동 (퀴즈는 "0/3 문제 · 탭해서 풀기") |
| 24 | 진행 중에서 **[ETF 시작 전 투자 퀴즈]** 탭 → O/X 3문제 풀기 | 정답 순서: **O → X → O** (ETF 분산·적금 금리·비상금) |
| 25 | 완료 | 취소선 + "완료! 포인트 적립됨" + **+60P 애니메이션 (12,450 → 12,510P)** |

## 5막 — 회고 (탭: 마이 → 기록)

| # | 조작 | 화면 |
|---|---|---|
| 26 | 마이 탭 | 소비 31% → (탭 전환) 저축 45% 게이지 → 투자 +12.7%, 요약 2열(아트 카드+리스트) |
| 27 | 캐러셀 스와이프·칩 전환 1~2회 | 9뷰 전환, 물잔·속도계·트리맵 차트 |

## 촬영 체크리스트

- **입력 문구 정확히**: `ETF 시도 해보고 싶어` (ETF 키워드 매칭)
- **퀴즈 정답**: O → X → O
- AI 타이핑 속도는 문장 길이에 비례(가변) — 버블 사이 호흡이 자연스러우니 끊지 말고 대기
- 씬 리셋: 해당 탭 새로고침 (상태 전부 초기화)
- 프로덕션: https://finmate-app-one.vercel.app
